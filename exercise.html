Exercise 1 — sum_to

Exercise 2 — fizzbuzz

Write a function fizzbuzz(n) that returns "Fizz" if n is divisible by 3, "Buzz" if divisible by 5, "FizzBuzz" if divisible by both, or the number as a string otherwise.
